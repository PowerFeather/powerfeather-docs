PowerFeather SDK docs are versioned. The latest preview is v2_dev; v1 is the current stable release on `main`.

## Choose a version
- [SDK v2 (latest preview)](./v2_dev/setup)

## Release flow
v2_dev -> v2 -> main. Main stays on v1.x.y until v2 is released.
