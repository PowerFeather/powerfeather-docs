PowerFeather SDK docs are versioned. The `2.x` docs track follows the current 2.x release line; `1.x` is the older stable release line.

:::note
SDK version and hardware revision are separate. PowerFeather-SDK 2.x supports both V1 and V2 ESP32-S3 PowerFeather boards. PowerFeather-SDK 1.x supports V1 boards only.
:::
